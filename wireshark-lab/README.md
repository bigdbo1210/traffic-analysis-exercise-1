# Wireshark Lab

## Objective
This lab focused on capturing and analyzing live network traffic using Wireshark.

## Traffic Captured
- HTTP traffic
- TCP traffic
- DNS traffic

## Observations
I observed multiple HTTP GET requests and TCP communications between my local system and external web servers. The captures showed how much background network activity occurs automatically when browsing websites.

## Security Concerns
The packet captures demonstrated how unencrypted HTTP traffic can expose data in plaintext. Attackers monitoring network traffic could potentially observe websites visited, IP addresses, and communication patterns.
