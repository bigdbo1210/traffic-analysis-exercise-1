# SOC Case Builder

## Overview

This project uses Python, APIs, JSON data, and object-oriented programming to organize cybersecurity investigation artifacts into cases.

## Features

* Pulls artifact data from an API
* Parses JSON records
* Creates Artifact objects
* Groups artifacts into SOC cases
* Calculates case severity
* Generates formatted case reports

## JSON Fields Discovered

* case_id
* indicator_type
* domain
* ip
* file_hash
* comment

## Technologies Used

* Python
* Requests
* JSON
* GitHub
* Object-Oriented Programming
